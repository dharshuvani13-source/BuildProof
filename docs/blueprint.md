# **App Name**: SkillProof

## Core Features:

- Secure User Authentication: Custom authentication system with email/password signup, OTP-based email verification, secure password hashing, session management via localStorage, and forgot password feature using OTP.
- Dynamic Profile Icon: Generates a profile icon from the first letter of the user's email, with a profile menu to view/edit details and logout.
- Skill Proof Submission: Authenticated users can submit a skill name, project title, description, and GitHub repository link. Uploaded screenshots are stored.
- Shareable Skill Proof Page: Generates a public Skill Proof page (accessible without login) displaying skill name, project evidence, description, GitHub link, and peer/mentor validation count.
- Peer Validation Counter: Users can increase the Peer Validation count on other people's proof of skill.
- Contact Us Form: A contact form allowing users to submit a message. Messages sent via the Contact Us page should go to a specified inbox.

## Style Guidelines:

- Primary color: Soft light blue (#A0D2EB) to evoke a sense of calmness and professionalism.
- Background color: Very light blue (#F0F8FF) with low saturation to complement the frosted glass cards.
- Accent color: Pale violet (#B0A0E0) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif.
- Use modern, minimalist icons suitable for professional web applications.
- Glassmorphism UI with frosted glass cards, subtle shadows, and rounded corners for a modern look. Clean, responsive layout for various devices.
- Smooth, subtle animations for transitions and interactive elements to enhance user experience.
